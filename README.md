# Advanced Snake Game

## Description
Advanced Snake Game is a responsive browser-based game developed using JavaScript, HTML5, and CSS3. The game automatically adjusts to different screen sizes and supports both desktop and mobile devices.

## Features
- Responsive game board that adapts to window size
- Three lives per game before Game Over
- Score and High Score tracking using browser local storage
- Multiple difficulty levels (Easy, Medium, Hard)
- Theme selection (Green, Blue, Purple)
- Sound effects with On/Off option
- Background music toggle
- Pause and Resume functionality
- Fullscreen mode support
- Optional Two-Player mode
- Touch swipe controls for mobile devices
- Random obstacles for added challenge

## Controls
### Keyboard
- Arrow Up: Move Up
- Arrow Down: Move Down
- Arrow Left: Move Left
- Arrow Right: Move Right

### Mobile
- Swipe Up, Down, Left, or Right on the game board.

## How to Run
1. Download the `snake_game.html` file.
2. Open the file in any modern web browser.
3. Select your preferred difficulty and theme.
4. Start playing immediately.
